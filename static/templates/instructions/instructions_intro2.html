On each trial, we will let you know how well you did by awarding you points for how accurate you were! The closer you place each square to its original location, the more points you will earn!

The task has four sections, each with 50 trials. In each section, there is a pattern to the order of the squares on the screen. This pattern is consistent for all the trials in the section, and learning it can make it easier to remember the locations of the squares. Be careful! The pattern will change after each section break.
Between each section, there will be a short break, and we'll remind you of the rules.

Please click "Start!" when you are ready to start the game.
